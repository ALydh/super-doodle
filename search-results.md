# Search: WH40K App "New Recruit"

## New Recruit — Army List Builder

**New Recruit** is a free, multi-platform army list builder for tabletop miniature games, with strong support for Warhammer 40,000 (10th Edition) and many other game systems.

- **Website:** https://www.newrecruit.eu/
- **Google Play:** https://play.google.com/store/apps/details?id=eu.newrecruit.www.twa
- **GitHub (BSData):** https://github.com/BSData/wh40k-10e

## Overview

New Recruit originated as a list builder exclusively for The 9th Age Fantasy Battles. It has since expanded to support many game systems including:

- Warhammer 40,000
- Age of Sigmar
- Horus Heresy
- Kill Team
- Warhammer: The Old World
- Warhammer Armies Project
- The Ninth Age

## Key Features

- Army list creation with automatic point counting and validation
- One-click list synchronization (no manual file handling)
- BattleScribe compatibility — imports `.ros` / `.rosz` roster files
- Game result tracking and statistics
- Free tournament organization tool
- Free to use with optional subscription for bonus features
- Data files sourced from the BSData community on GitHub

## Reception

- 190K+ downloads, averaging ~390/day recently
- Praised for Crusade Order of Battle tracking and clean UI
- Some criticism about the interface being confusing for new players (e.g., adding units is not immediately obvious)
- Gaining traction particularly in Europe as a solid BattleScribe alternative
